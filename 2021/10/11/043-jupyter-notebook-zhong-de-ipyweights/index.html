jupyter notebook中交互式控件相关

一、需求

在jupyter notebook（jn）中实现的功能依靠ipyweights包可以实现交互效果，更加直观，更可以形成APP发布

二、介绍

传统上，每次需要修改笔记本单元格的输出时，都需要更改代码并重新运行受影响的单元格。这可能很繁琐、低效甚至容易出错，对于非技术用户来说，甚至是不切实际的。这就是ipywidgets发挥作用的地方：它们可以嵌入到笔记本中，并提供一个用户友好的界面来收集用户输入并查看更改对数据/结果的影响，而不必与代码交互。

官方介绍：

ipywidgets, also known as jupyter-widgets or simply widgets, are interactive HTML widgets for Jupyter notebooks and the IPython kernel.

Notebooks come alive when interactive widgets are used. Users gain control of their data and can visualize changes in the data.

Learning becomes an immersive, fun experience. Researchers can easily see how changing inputs to a model impact the results.

三、使用

说明书上的东西不做介绍，仅介绍使用过程中的新接触的东西

  1. 控件字体（style）调整

    每一种控件都可以通过 控件.style.keys 来知道它有哪些属性，然后通过style来统一定义使用的时候会更加方便一点，以button为例

    style = dict(
    button_color='yellowgreen',
    font_style='italic',
    font_weight='bold',
    font_variant="small-caps",
    text_color='red',
    text_decoration='underline'
)
image_btn = Button(description='显示卫星影像', layout=Layout(width='auto',height='auto') ,style=style)

    对于像dropdown这种下拉列表的description没有找到修改字体的属性，可以直接用HTML，控件中所有的字体都可以直接使用HTML控制显示格式，就是麻烦一点

    dropdpwn.description = 
    黄河口悬浮泥沙分布在线反演APP
    "

  2. 布局

    weights中提供了很多种排列方式，最开始我一直用默认的布局，只用了最简单的HBox和VBox这两种水平和竖直排列，其中的参数基本也没研究，经过简单的布局，就形成了下图一所示的布局。

    其中，标题和底下的介绍都是直接用markdown语法写的，然后摆上一个地图map，下面的控件都是水平垂直的堆叠。

    个人觉得看起来也还算顺眼，但有如下问题：

    （1）markdown写的东西和底下用控件写的无法一块布局，只能依次堆叠

    （2）地图太宽，实际只用中间的一部分

    （3）因为是通过网页显示的，所以默认的大小控件用多了就会超出一页，让我看着很不舒服

    于是研究了一番，对布局做成了如下图所示，一方面缩小了布局，另一方面gee app基本的布局都是这样的，利用jn制作app我也就沿用这种布局了

    主要用到GridBox这个类，可以方便的控制每部分的大小（其实也不是很方便，我记得之前用winform编可视化界面，只需要拖动控件然后拉动就可以调节大小）；然后就是灵活运用HTML，任何字符的显示不但可以用控件本身的属性进行控制也可以用HTML进行根本上的变化；最后就是同一类控件中属性的统一编写，方便统一调用。下面依次说明

    • GridBox

      The GridBox class is a special case of the Box widget.

      The Box widget enables the entire CSS flexbox spec, enabling rich reactive layouts in the Jupyter notebook. It aims at providing an efficient way to lay out, align and distribute space among items in a container.

      Again, the whole grid layout spec is exposed via the layout attribute of the container widget (Box) and the contained items. One may share the same layout attribute among all the contained items.

      上面是官方的解释，在官方实例文档Widget Styling中有详细的解释，我对它的理解就是给你一个box，你可以随便用若干个格子对box进行分隔，然后给每个控件几个格子的大小由你定

      header  = Button(description='Header',
                 layout=Layout(width='auto', grid_area='header'),
                 style=ButtonStyle(button_color='lightblue'))
main    = Button(description='Main',
                 layout=Layout(width='auto', grid_area='main'),
                 style=ButtonStyle(button_color='moccasin'))
sidebar = Button(description='Sidebar',
                 layout=Layout(width='auto', grid_area='sidebar'),
                 style=ButtonStyle(button_color='salmon'))
footer  = Button(description='Footer',
                 layout=Layout(width='auto', grid_area='footer'),
                 style=ButtonStyle(button_color='olive'))

GridBox(children=[header, main, sidebar, footer],
        layout=Layout(
            width='50%',
            grid_template_rows='auto auto auto',
            grid_template_columns='25% 25% 25% 25%',
            grid_template_areas='''
            "header header header header"
            ". main . sidebar "
            "footer footer footer footer"
            ''')
       )

      上面这段代码就很直观的解释了，grid_template_rows=’auto auto auto’, grid_template_columns=’25% 25% 25% 25%’

      这两句话就是将box分成三行，每行长短自动调整，每列都占25%，然后grid_template_areas这个参数就负责每个控件占的大小：第一行header占4个25%，第二行：.表示没有，就是空25%，然后main和sider分别占25%，第三行同理

    • HTML

      所有字符包括中英文，都可以用HTML语言进行控制。包括

      labe11 = HTML(value="<p><font size='2.5' face = '楷体'> 1. 通过云量筛选指定日期黄河口的影像<br />\
              2. 利用内置的悬沙反演算法将选择的影像执行反演程序，显示反演结果 <br / >\
              3. 制作遥感影像和反演结果的gif动图<br / >\
              4. 叶绿素浓度的反演影像以及海表风速的显示以及下载<br / >\
              Contact： <a href='https://suxiaoming.tech/'>Xiaomingbluesky</a></font></p>")

      上面这种就是直接输出为HTML块，可以当作HTML控件进行布局

      image_select = widgets.Dropdown(
    description = '选择数据源：',
    options=['Landsat','MODIS'],
    value = 'Landsat',
    style = style
)
image_select.description = "<font size='2' face = '华文细黑'>选择数据源：</font>"

      这种就是利用控件的description属性进行控制，也算比较方便

    • keys

      每个控件的外观都有自己的属性，可以通过 控件.style.keys进行查看

      bt_style = dict(
    button_color='#9ACD32',
    font_style='italic',
    font_weight='bold',
    font_variant="small-caps",
    text_color='red',
    text_decoration='underline'
)

      上面就是将Button的属性通过定义一个字典，将各种属性存在里面，可以直接在每个控件中的style属性中调用。

写在最后，很多问题用百度是百度不出来的，有时候用谷歌一下就能找到，即使不懂用英语怎么写，就用最简单的“how can i use widgets button in jupyter notebook”，也可以获得意想不到的收获。
